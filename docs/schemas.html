Schemas

Canonical JSON Schemas are located in the repository under /schemas. Key schemas:

  • manifest.schema.json
  • agent.schema.json
  • workflow.schema.json
  • data-model.schema.json
  • api-routes.schema.json
  • ui-components.schema.json
  • deployment.schema.json

Schema metadata: Each schema includes self-describing metadata for versioning and tool compatibility:

{
  "$id": "https://vibespec.vibecodeunited.com/schema/<file>.schema.json",
  "$schema": "https://json-schema.org/draft/2020-12/schema",
  "title": "Descriptive Title",
  "vibespecVersion": "0.1.0"
}
  • $id is the canonical, unique URL for the schema.
  • $schema is always https://json-schema.org/draft/2020-12/schema.
  • title names the schema (e.g., Manifest, Agent, Workflow).
  • vibespecVersion matches VERSION.json for auto-discovery and compatibility checks.

Use npm install and npm run validate to verify example projects.
